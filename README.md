# MathKids - Jeu de Maths pour Enfants

Application Android pour apprendre les mathematiques en s'amusant, destinee aux enfants de 5 ans.

## Fonctionnalites

### Compter (1-10)
Apprends a compter en identifiant le nombre d'objets affiches a l'ecran. Des emojis colores rendent l'apprentissage visuel et amusant.

### Addition
Resous des additions simples (nombres de 1 a 5) avec une representation visuelle pour comprendre le concept.

### Soustraction
Decouvre la soustraction avec des exemples visuels montrant les elements retires.

### Reconnaissance de Formes
Identifie les formes geometriques : cercle, carre, triangle, rectangle, etoile et losange.

## Caracteristiques
- Interface coloree et adaptee aux enfants
- Animations ludiques (bounce, shake, pop-in)
- Retour sonore pour les bonnes et mauvaises reponses
- Systeme de score avec etoiles
- 10 questions par partie
- Ecran de resultats avec option de rejouer
- Support du francais

## Technologies
- Kotlin
- Android SDK (minSdk 24, targetSdk 34)
- Material Design Components
- ViewBinding

## Construction
Ouvrir le projet dans Android Studio et lancer la compilation avec Gradle.

```bash
./gradlew assembleDebug
```
